## 📌 About This Repository

This repository is a **structured roadmap to learn Python for Data Science**.
It covers everything from writing your first line of Python to working with real-world datasets — through **lecture-wise notebooks** with hands-on examples and coding practice.

**Who is this for?**
- 🙋 Absolute beginners with zero programming experience
- 🔁 Developers transitioning into Data Science
- 📊 Anyone looking to build a solid Python foundation for AI/ML

---

## 🎯 What You'll Learn

- ✨ Python Fundamentals
- ✨ Control Flow & Logic Building
- ✨ Data Structures
- ✨ Object-Oriented Programming
- ✨ NumPy & Pandas
- ✨ Data Visualization

<summary><h3>📗 Lecture 2 &nbsp;—&nbsp; Operators, Logic & Modules &nbsp; <code>✅ Completed</code></h3></summary>

<br/>

> Covers decision-making, loops, and Python's powerful standard library modules.

**⚙️ Operators**

| Category | Operators | Purpose |
|----------|-----------|---------|
| Arithmetic | `+ - * / % // **` | Mathematical calculations |
| Relational | `== != > < >= <=` | Comparing values |
| Logical | `and or not` | Combining conditions |
| Bitwise | `& \| ^ ~ << >>` | Bit-level operations |
| Assignment | `+= -= *= /= ...` | Shorthand variable updates |
| Membership | `in`, `not in` | Checking existence in sequences |
| Identity | `is`, `is not` | Comparing object identity |

**🔀 Control Flow**

| Concept | Description |
|---------|-------------|
| `if / else` | Binary decision making |
| `elif` ladder | Multi-branch conditions |

**🔁 Loops**

| Concept | Description |
|---------|-------------|
| `while` loop | Repeat while condition is true |
| `while` with `else` | Execute block after loop ends normally |
| `for` loop | Iterate over sequences |
| 🎯 Mini Project | **Guessing Game** — number guessing with loop logic |

**📦 Standard Library Modules**

| Module | Key Use |
|--------|---------|
| `math` | Mathematical functions (`sqrt`, `ceil`, `pi`) |
| `keyword` | List all Python reserved keywords |
| `random` | Generate random numbers and choices |
| `datetime` | Work with dates, times, and timedeltas |
